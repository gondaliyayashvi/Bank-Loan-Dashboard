# 📊 Bank Loan Analysis Dashboard
## 📌 Project Overview

This project demonstrates a Bank Loan Analysis Dashboard developed using Power BI and SQL queries, with data sourced from financial_loan.csv. The dashboard helps financial institutions analyze loan applications, repayment patterns, borrower demographics, and portfolio health.

The objective is to provide data-driven insights into loan performance, borrower risk assessment, and regional/segment-wise distribution of loans, enabling banks to make smarter lending decisions.

## 🛠️ Technologies Used

MS SQL Server – For data storage and query execution

SQL Queries – For data extraction, transformation, and KPI calculations (Query Doc.docx)

Power BI – For interactive visualizations (Bank Loan PPT Power BI.pptx)

CSV Dataset – financial_loan.csv (loan data used for analysis)

Documentation – Terminologies and query details (Terminologies in Data.docx, Query Doc.docx)

## 📊 Key Features
### 🔹 Summary Dashboard

Total Loan Applications (Overall, MTD, PMTD)

Total Funded Amount & Total Amount Received

Average Interest Rate (Overall, MTD, MoM)

Average Debt-to-Income Ratio (DTI)

Good Loan vs Bad Loan KPIs

Loan Status Grid View

### 🔹 Overview Dashboard

Monthly Trends (loan issuance over time)

Regional Analysis by state

Loan Term Analysis (distribution of 36 vs 60 months)

Employment Length Analysis (impact of borrower stability)

Loan Purpose Breakdown (education, debt consolidation, etc.)

Home Ownership Analysis

### 🔹 Detailed Dashboard

Comprehensive grid with borrower profiles, loan status, repayment, and risk metrics

## 📈 Insights & Use Cases

Identify Good Loans vs Bad Loans and their contribution to overall portfolio health

Evaluate regional loan demand and repayment behavior

Analyze borrower stability using employment length and home ownership

Track loan purposes and tailor lending strategies accordingly

Monitor interest rates and DTI ratios for risk assessment
